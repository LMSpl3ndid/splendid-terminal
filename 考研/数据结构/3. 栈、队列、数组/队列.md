# 2.循环队列
为了解决队列在出入过程中队首指针置于申请空间尾部而导致队尾无法进队（假溢出）

判队空：front=\=rear
判队满：(rear+1)%Maxsize=\=front 

入队：rear=(rear+1)%Maxsize
出队：front=(front+1)%Maxsize    **（注意队首加一**
