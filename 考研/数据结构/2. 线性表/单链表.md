## 1. 定义

大结构：头结点+其他结点
结点：数据域+指针域

（头结点数据域没有内容或表长）
## 2. 特点
优点：增删无需移动元素
缺点：不能随机存取

头结点优点：
	链表第一个位置操作与其他位置操作统一
	空表和非空表操作得到统一
## 3. 基本操作
### 3.1 建表（头插法、尾插法）

| 头插法  | 尾插法  |
| ---- | ---- |
| O(n) | O(n) |

### 3.2 查找（按序号、按值）

| 序号   | 值    |
| ---- | ---- |
| O(n) | O(n) |
### 3.3 插入（后插、前插）
i位置后插a: 找到i-1，插入a
	先把插入节点的后指针处理好，在处理原表
i位置前插a: 找到i，后插
	(1) 按后插连接好指针域
	(2) 再交换前后结点数据域

| 前插   | 后插   |
| ---- | ---- |
| O(n) | O(n) |

### 3.4 删除结点
承接+free
O(n)

### 3.5 求表长
O(n)
