# 比较！

|         | 顺序表                         | 链表           |
| ------- | --------------------------- | ------------ |
| 存取方式    | 顺序or随机                      | 顺序           |
| 逻辑和物理结构 | 逻辑相 = 物理相邻                  | 逻辑相邻 != 物理相邻 |
| 增删查     | 下标访问O(1)                    |              |
| 空间分配    | 静态存储: 浪费和溢出   动态存储: 扩充时大量移动 | 便于分配，密度小     |


|     | 顺序表    | 链表     |
| --- | ------ | ------ |
| 存储  | 静态，密度高 | 动态，密度低 |
| 运算  | 访问优势   | 增删优势   |
| 环境  |        |        |
